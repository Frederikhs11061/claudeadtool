Install

One-command install (Unix/macOS/Linux):

curl -fsSL https://raw.githubusercontent.com/Frederikhs11061/claudeadtool/main/install.sh | bash

Or clone and run install:

git clone https://github.com/Frederikhs11061/claudeadtool.git
cd claudeadtool
./install.sh

Quick start in Claude Code
