Claude Ads

Paid advertising audit & optimization skill for Claude Code. Covers Google Ads, Meta, YouTube, LinkedIn, TikTok & Microsoft Ads with 190 audit checks and industry templates.

Claude Code Skill MIT

Install

One-command install (Unix/macOS/Linux):

curl -fsSL https://raw.githubusercontent.com/Frederikhs11061/claudeadtool/main/install.sh | bash

Or clone and install manually:

git clone https://github.com/Frederikhs11061/claudeadtool.git
cd claudeadtool
./install.sh

Quick start

In Claude Code:

  • /ads audit — full multi-platform audit
  • /ads google — Google Ads
  • /ads meta — Meta Ads
  • /ads linkedin — LinkedIn Ads
  • /ads plan saas — strategic plan (e.g. saas, ecommerce)
